## Importing the bike-share data

All the csv files for 2023 are imported and combined together to create
the data frame bike_share_2023. list.files produces a character vector
of all the relative file paths in the folder *bike_share_csv_files*.
lapply applies the function read_csv to the file paths in the vector
bind_rows combines all the rows into one data frame.
